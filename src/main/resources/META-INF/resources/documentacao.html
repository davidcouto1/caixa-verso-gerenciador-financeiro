Guia completo da API de Simulação de Financiamentos

📋 Sobre o Projeto

API REST desenvolvida em Java 25 com Quarkus 3.35.4 para simulação de financiamentos utilizando cálculo de juros compostos.

O sistema persiste os dados em banco de dados H2 embutido (file-based) e oferece documentação automática via OpenAPI/Swagger.

✨ Características Principais

  • ✅ Cálculo preciso usando BigDecimal com 2 casas decimais
  • ✅ Memória de cálculo detalhada mês a mês
  • ✅ Validação de entrada com Bean Validation
  • ✅ Persistência em H2 Database
  • ✅ Arquitetura em camadas (Resource → Service → Repository)
  • ✅ Tratamento de erros padronizado
  • ✅ Cobertura de testes ≥ 80% (Jacoco)
  • ✅ 100% nativo - sem Docker

🔌 Endpoints da API

POST /api/simulacoes

Criar nova simulação

Cria uma simulação de financiamento calculando os juros compostos mês a mês.

Request Body: 
{
  "valorInicial": 1000.00,
  "taxaJurosMensal": 1.5,
  "prazoMeses": 12
}
Response (201 Created): 
{
  "id": 1,
  "valorInicial": 1000.00,
  "taxaJurosMensal": 1.5,
  "prazoMeses": 12,
  "valorTotalFinal": 1195.62,
  "valorTotalJuros": 195.62,
  "memoriaCalculos": [
    {
      "mes": 1,
      "saldoInicial": 1000.00,
      "juro": 15.00,
      "saldoFinal": 1015.00
    },
    ...
  ]
}
GET /api/simulacoes/{id}

Buscar simulação por ID

Retorna uma simulação existente com todos os detalhes e memória de cálculo.

Exemplo: 
GET /api/simulacoes/1
Response (200 OK): 
{
  "id": 1,
  "valorInicial": 1000.00,
  "taxaJurosMensal": 1.5,
  "prazoMeses": 12,
  "valorTotalFinal": 1195.62,
  "valorTotalJuros": 195.62,
  "memoriaCalculos": [...]
}
GET /api/simulacoes/health

Health Check

Verifica se a API está funcionando corretamente.

Response (200 OK): 
Simulador de Financiamentos - API is running!

🧮 Fórmula de Cálculo

O sistema utiliza a fórmula de juros compostos:

M = C × (1 + i)^n

Onde:
  M = Montante final
  C = Capital inicial (valorInicial)
  i = Taxa de juros (taxaJurosMensal / 100)
  n = Número de períodos (prazoMeses)

Exemplo prático:

Valor Inicial: R$ 1.000,00
Taxa: 1,5% ao mês
Prazo: 12 meses

M = 1000 × (1 + 0.015)^12
M = 1000 × 1.195618...
M ≈ R$ 1.195,62

Total de Juros: R$ 195,62

⚠️ Regras de Validação

  • ✅ Valor Inicial: Deve ser maior que zero
  • ✅ Taxa de Juros: Deve ser maior que zero
  • ✅ Prazo: Deve estar entre 1 e 360 meses

Resposta de erro (400 Bad Request):

{
  "status": 400,
  "error": "Bad Request",
  "message": "O valor inicial deve ser maior que zero",
  "path": "/api/simulacoes"
}

🧪 Testando a API

1. Via Swagger UI (Interativo)

Abrir Swagger UI

2. Via cURL (Terminal)

# Health Check
curl -X GET http://localhost:8080/api/simulacoes/health

# Criar Simulação
curl -X POST http://localhost:8080/api/simulacoes \
  -H "Content-Type: application/json" \
  -d '{"valorInicial": 1000.00, "taxaJurosMensal": 1.5, "prazoMeses": 12}'

# Buscar Simulação
curl -X GET http://localhost:8080/api/simulacoes/1

💾 Banco de Dados

O projeto utiliza H2 Database embutido (file-based):

  • 📁 Arquivo: ./data/simulador-db
  • 🔄 Schema: Criado automaticamente pelo Hibernate
  • 🗃️ Tabelas: simulacao e memoria_calculo
  • 🔗 Relação: 1:N (uma simulação tem várias memórias de cálculo)

🚀 Como Executar

# 1. Executar em modo de desenvolvimento
mvn quarkus:dev

# 2. Compilar o projeto
mvn clean compile

# 3. Rodar todos os testes
mvn test

# 4. Gerar relatório de cobertura (Jacoco)
mvn clean test
# Ver relatório em: target/site/jacoco/index.html
