🏦 Simulador de Financiamentos

API REST para cálculo de juros compostos

📊 Cálculo Preciso

Utiliza BigDecimal para cálculos precisos de juros compostos mês a mês

💾 Persistência

Armazena simulações em banco H2 embutido para consultas futuras

📝 Memória de Cálculo

Detalhamento completo da evolução dos juros mês a mês

✅ Validação

Bean Validation para garantir dados consistentes
